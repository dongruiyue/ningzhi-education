# 学习路径与课程

## 完整学习路径

```
入学测试 → 分班 → 阶段学习 → 期中模考 → 查漏补缺 → 冲刺模考 → 考试
                    ↑                                              ↓
                    └────────── 全程督学跟踪 ──────────────────────┘
```

## 第一步：入学测试

到校参加线下入学测试，全面摸底当前英语水平。这是后续学习效果的 **benchmark**（基准线），也是分班依据。

## 第二步：分班与目标设定

根据测试结果，分配到合适的班级，设定阶段性目标分数。

## 第三步：阶段学习

按照真经教学体系，听说读写四科同步推进。每个阶段包含：
- 正课教学
- 课后练习
- 每日打卡
- 老师反馈

## 第四步：期中模考

检测阶段性进步，及时调整教学策略。

## 第五步：冲刺模考

考前密集模考，适应考试节奏，查漏补缺。

## 课程类型

| 班型 | 适合人群 | 特点 |
|------|---------|------|
| 一对一 | 时间灵活、需要个性化方案 | 最高关注度 |
| 小班（3-6人） | 大多数学生 | 有竞争有氛围 |
| 冲刺班 | 考前1-2个月 | 密集模考+讲评 |

## 督学服务

- 每日学习任务打卡
- 每周进度报告
- 老师微信随时答疑
- 模考成绩跟踪
