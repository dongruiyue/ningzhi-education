# 学习路径与课程

## 完整学习路径

```
入学测试 → 水平定位 → 教材匹配 → 阶段学习 → 阶段测试 → 调整方案 → 考前冲刺 → 考试
                ↑                                                              ↓
                └──────────────── 全程家长沟通 ────────────────────────────────┘
```

## 第一步：入学测试

到校参加英语水平测试，全面了解孩子当前听说读写能力。测试结果 + 家长期望 = 教材匹配方案。

## 第二步：教材匹配

根据测试结果，从五套教材中选出最适合的组合方案。家长会收到详细的**教材选择建议**和**学习规划表**。

## 第三步：阶段学习

按选定教材体系推进，每阶段3-6个月：
- 每周1-2次课
- 课后每日打卡
- 阶段测试检查

## 第四步：考前冲刺（如有考试需求）

- KET/PET全真模考
- 题型专项训练
- 考前心理疏导

## 课程类型

| 班型 | 适合人群 | 特点 |
|------|---------|------|
| 一对一 | 有明确目标/时间紧张 | 效率最高，个性化最强 |
| 小班（2-4人） | 大部分学生 | 有同学互动，互相促进 |
| 常规班（4-6人） | 长期素养提升 | 氛围好，性价比高 |
