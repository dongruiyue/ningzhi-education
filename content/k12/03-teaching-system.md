# 教学体系 — 校内提升

## 我们的教学理念

**能力驱动分数，而非分数驱动能力。**

很多机构的做法是"考什么教什么"——KET考什么题型就练什么题型。短期看起来有效，但能力没有真正提升，到了PET/FCE阶段就会遇到瓶颈。

我们的理念正好相反：**用国际教材体系真正提升英语能力，考试出分只是水到渠成的结果。**

## 课堂结构

每节课（90分钟）包含四个环节：

1. **热身/复习**（10分钟）：回顾上节课内容，检查作业
2. **核心教学**（40分钟）：新内容讲解与互动练习
3. **应用输出**（25分钟）：学生开口说、动手写，老师即时反馈
4. **总结/任务布置**（15分钟）：本课小结，下周任务布置

## 课下跟进

- 每日朗读打卡
- 每周词汇测试
- 定期写作批改
- 家长群进度通报

## 学习效果保障

- 入学测试 → 建立基准
- 阶段测试 → 可视化进步
- 考前模考 → KET/PET全真模拟
- 出分后 → 规划下一阶段
