# 教材选择定位体系

## 五套教材概览

宁致教育是深圳少数同时拥有五套国际英语教材体系的机构。每套教材各有侧重，我们根据学生的年龄、目标和当前水平进行组合推荐。

## 教材对比表

| 教材 | 背景 | 核心侧重 | 适合水平 | 对应考试 |
|------|------|---------|---------|---------|
| **PowerUp** | 剑桥大学出版社 | 少儿英语启蒙，听说读写综合 | 入门-初级 | Pre A1 Starters / A1 Movers |
| **UNlock** | 剑桥大学出版社（英式） | 批判性思维+学术英语，听说&读写分册 | 初级-中级 | KET / PET |
| **Pathways** | National Geographic（美式） | 学术英语+全球视野，听说&读写分册 | 中级-中高级 | PET / FCE |
| **Reading Explorer** | National Geographic | 阅读理解+词汇积累，非虚构类文本 | 初级-中高级 | 全阶段适用 |
| **Great Writing** | National Geographic | 写作专项，从句子到段落到文章 | 初级-中高级 | 全阶段适用 |

## 教材选择定位矩阵

根据学生的**年龄、目标、当前水平**三个维度，推荐教材组合：

### 维度一：按年龄

| 年龄段 | 推荐主教材 | 搭配 |
|--------|-----------|------|
| 6-8岁 | PowerUp | + Reading Explorer Foundation |
| 8-10岁 | UNlock Basic | + Reading Explorer L1 |
| 10-12岁 | UNlock L1-L2 | + Great Writing L1 |
| 12-15岁 | Pathways L1-L2 | + Reading Explorer L2+ / Great Writing L2+ |

### 维度二：按目标

| 目标 | 推荐组合 |
|------|---------|
| 备考KET | PowerUp (后期) → UNlock L1 + Reading Explorer L1 |
| 备考PET | UNlock L2 → Pathways L1 + Great Writing L1 |
| 校内提分 | UNlock (同步校内) + Great Writing (写作专项) |
| 综合素养提升 | Reading Explorer (阅读) + UNlock 或 Pathways (听说写) |

### 维度三：按当前水平

| 入学水平 | 起点教材 |
|---------|---------|
| 零基础/刚接触 | PowerUp Starter |
| 能简单对话 | UNlock Basic 或 PowerUp L3 |
| 能基本交流 | UNlock L1 或 Pathways Foundations |
| 能流利交流 | Pathways L1+ |

## 一个学生的全周期案例

> 小明，6岁入门：
> PowerUp L1-L3 (6-8岁) → UNlock Basic-L1 (8-10岁) → Reading Explorer L1-L2 (同步) → Pathways L1 + Great Writing L1 (10-12岁) → 备考PET

每个阶段的教材衔接都是平滑的，能力递进清晰可见
